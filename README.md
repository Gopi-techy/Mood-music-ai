# Mood Music AI

An open-source AI-powered application that curates and plays music based on your mood.

## Features
- Detect or select your current mood.
- Automatically curated music recommendations.
- Local graphical user interface (`mood_music_gui.py`).
- Simple setup with environment variables (`.env`).

## Setup

1. Copy `.env.example` to `.env` and fill in your API keys / configuration.
2. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the GUI:
   ```bash
   python mood_music_gui.py
   ```
